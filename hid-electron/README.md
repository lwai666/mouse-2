node_module/electron/dist下有electron.app ? 如果没有就没有安装成功，pnpm下一次没有安装成功比较麻烦，后续安装该版本都不成功，可以查一下pnpm如何清理缓存

### Install

```bash
$ pnpm install
```

### Development

```bash
$ sudo pnpm run dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
